myMovies

The web application that will provide users with access to information about movies, directors, and genres.

Request URL - Endpoint Query Parameter HTTP Method Request Data Format Response
Return a list of all movies to the user /movies None GET None A JSON object holding data about all the movies
Return data about a single movie by title to the user /movies/:title title GET None A JSON object holding data about a single movie based on the title passed in the url containing (description, genre, director, image Url)
Return data about a genre by name/title /genre/:genreName genreName GET None A JSON object of movies based on genre passed in the url
Return data about a director by name /director/:directorName directorName GET None A JSON object of movies based on director name passed in the url
Allow new users to register themselves /users None POST A JSON object about a user to be registered A JSON object about the user that is added with user id
Allow users to update their info /users/:username id PUT None A JSON object about the updated userName that is added with user id
Allow users to add movie to their favorites /users/:username/movies/:title movieTitle POST None Returns a success message that the movie added to favorites
Allow users to remove movie from their favorites /users/:username/movies/:title movieTitle DELETE None Returns a success message that the movie removed from favorites
Allow users to deregister /users/:username None DELETE None Returns a success message that the user is removed
